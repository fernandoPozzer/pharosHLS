# PharosHLS

**Developed by** Fernando Pedrazzi Pozzer

**Oriented by** prof. Dr. Mateus Beck Rutzig

**Cooriented by** Renan Carlos Gomes de Farias

PharosHLS is a tool designed to assist in understanding the behavior of functions when
implemented on FPGAs. By analyzing hardware synthesis metrics and characteristics, the tool
provides insights that enable developers to assess the impact of different design choices, thereby
supporting more informed and efficient decision-making throughout the development process
of reconfigurable hardware-based systems.

This Python library was developed by Fernando Pedrazzi Pozzer as part of his bachelor's thesis in Computer Science at the Federal University of Santa Maria (UFSM).
